# Generated Terraform

This directory contains Terraform configurations generated by the policy-as-code adapters.

## Structure

```
generated/terraform/
├── aws/
│   └── {account}/
│       └── {policy-name}.tf
├── gcp/
│   └── {project}/
│       └── {policy-name}.tf
├── azure/
│   └── {subscription}/
│       └── {policy-name}.tf
├── paloalto/
│   └── {device-group}/
│       └── {policy-name}.tf
├── fortinet/
│   └── {adom}/
│       └── {policy-name}.tf
└── illumio/
    └── {pce}/
        └── {policy-name}.tf
```

## Important Notes

- These files are auto-generated. Do not edit manually.
- Changes to these files are made by modifying policies in `policies/` or objects in `registry/`.
- Each file is tagged with the source policy and commit that generated it.
